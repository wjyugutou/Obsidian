## 创建表
```sql
CREATE TABLE `user` (
  `id`         BIGINT UNSIGNED NOT NULL AUTO_INCREMENT,
  `username`   VARCHAR(50)     NOT NULL,
  `email`      VARCHAR(100)    DEFAULT NULL,
  `password`   VARCHAR(255)    NOT NULL COMMENT 'bcrypt hash',
  `phone`      VARCHAR(20)     DEFAULT NULL,
  `avatar`     VARCHAR(500)    DEFAULT NULL,
  `status`     TINYINT UNSIGNED NOT NULL DEFAULT 1 COMMENT '0=禁用 1=启用',
  `created_at` DATETIME        NOT NULL DEFAULT CURRENT_TIMESTAMP,
  `updated_at` DATETIME        NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_username` (`username`),
  UNIQUE KEY `uk_email` (`email`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

- **`BIGINT UNSIGNED`**：使用无符号大整数，范围 0 ~ 18446744073709551615（约1844亿亿），几乎永远用不完。**不要用 INT**，互联网项目数据量增长远超预期。
- **`AUTO_INCREMENT`**：自动递增，插入时无需手动指定 id。
- **`NOT NULL`**：主键必须非空。
- **`PRIMARY KEY`**：显式声明为主键，InnoDB 会基于它建立聚簇索引。
